# java-classes

5 simple Java classes with attributes and methods for practice.

## Classes

1. **Tree.java** - Tree with height, type attributes and grow, shedLeaves methods
2. **Charger.java** - Charger with brand, voltage, isConnected attributes and connect, disconnect methods  
3. **TV.java** - TV with brand, channel, volume, isOn attributes and turnOn, turnOff, changeChannel methods
4. **WaterBottle.java** - WaterBottle with material, capacity, waterLevel attributes and fill, drink methods
5. **Bird.java** - Bird with species, color, canFly attributes and fly, eat, makeSound methods

## Purpose
This repository is created for learning basic Java class structure with attributes and methods.
